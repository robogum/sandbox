## メルカリで本を売るので、その値段を決める(6)  
サードパーティを使ってGoogle検索をする。

Google検索は、スクレイピングが禁止されているので、Google検索のサードパーティを使って検索します。
Google検索はいつくかありますが、次の３つを使ってみました。
「VALUE SERP」「Serp Api」「scrape.do」
どれでも、方法は大体一緒なので、後は、料金などでどれを使うかを検討します。

上記は、Google検索を行うサービスですが、Amazonの物もあるようです。
いつか、扱う事があるかもしれませんが、メモっておきます。（forest）

### VALUE SERP
アカウントを作成して、API Keyを貼り付ければOKです。
登録時に、100回まで利用できる。Pay-As-You-Go：$2.50 per 1k.
注意）使用する前に、料金を確認してください。

[サンプルコード VALUE SERP](https://github.com/robogum/sandbox/blob/main/blog/projects/project01/code/valueserp.ipynb)


### Serp Api
アカウントを作成して、API Keyを貼り付ければOKです。
クレジットの番号を入力する必要はありません。
アカウントを作成すると、250回/月 使用する事ができます。
注意）使用する前に、料金を確認してください。

[サンプルコード Serp Api](https://github.com/robogum/sandbox/blob/main/blog/projects/project01/code/serpapi.ipynb)


### scrape.do
アカウントを作成して、tokenを下で作成したコードに貼り付ければＯＫです。クレジットの番号は入力する必要ありません。
アカウントを作成すると、1000クレジット利用可能になるので、100回は利用できます。
注意）使用する前に、料金を確認してください。

[サンプルコード scrape.do](https://github.com/robogum/sandbox/blob/main/blog/projects/project01/code/scrape_do.ipynb)
