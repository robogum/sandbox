## メルカリで本を売るので、その値段を決める(7)  
ISBNコードから、本の値段を検索する。

ISBNコードから、一度、本のタイトルを検索して、
その結果（タイトル）から、Google検索をして、現在の本の相場を表示します。


（１）「ISBNコードから本のタイトルを検索する。」を行います。
ここでは、「国会図書館」と「OpenBD」の両方を使用します。
最初に「Open BD」を使って、ISBNで検索を行います。
結果がなかった時は、「国会図書館」を使って、ISBNで検索を行います。

（２）「サードパーティを使ってGoogle検索をする。」を行います。
上記（１）の結果、本のタイトルを検索します。
3つの使用例を出したので、お好みのサードパーティを利用して検索をします。

[ISBN検索](https://github.com/robogum/sandbox/blob/main/blog/projects/project01/code/searchbook.ipynb)

ターミナルからは
```

> cd project01/code
> python -m searchbook 9784844359357 -t valueserp

```
