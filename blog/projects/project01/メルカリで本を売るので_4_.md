## メルカリで本を売るので、その値段を決める(4)  
何を検索するのか。

そもそも、Google検索でどのように検索すれば、本の値段が出てくるのかを試してみる必要があります。
Googleを表示して、「ISBN {本のISBNコード} 価格」
で検索してみます。

```
一致する情報はみつかりませんでした。
```

マイナーな本だったためでしょうか。Googleしても一致する物はありませんでした。
「ISBN {本のISBNコード}」 （価格をとって検索してみます。）

```
（5件、表示されました。）
```

５件表示されましたが、本のタイトルは正しく出てきましたが、価格はよくわかりません。
これですと、本を売る値段を決めるための、何の参考にもなりません。

どう検索すれば良いかは、これはこれで、結構、奥が深くなってしまうので、
「{本のタイトル} 中古 価格」で検索する事にします。
「メルカリで検索すればいいじゃないか。」とか、色々ありますが、今は、Google検索で、「{本のタイトル} 中古 価格」で検索します。

```
（結構、それなりに、検索できました。メルカリだと、800円、他だと、220円から500円位だということが判りました。）
```

ということで、なんとなく出てくることが判ったので、これで進めます。
ここまでは、ブラウザでGoogle検索を手入力でやっていますが、これをスクレイピングで行います。
Googleは、スクレイピングができないので、サードパーティを使って行います。

ISBNから直接検索できないのが、いまいちなので、そこは後で何とかしましょう。
まずは、本のタイトルから検索します。
